    Android Installation from QR codes

    Development version

    Dependencies: python, qt, pyqt, gettext, ecdsa, slowaes. On Ubuntu you can install the dependencies with:

      sudo apt-get install python-pip python-qt4 pyqt4-dev-tools python-setuptools
  sudo pip install ecdsa slowaes

    Check out the code from Github

      git clone git://github.com/spesmilo/electrum.git

    Compile the icons

      pyrcc4 icons.qrc -o gui/icons_rc.py

    Create translations

      python mki18n.py

    Install Electrum

      sudo python setup.py install
